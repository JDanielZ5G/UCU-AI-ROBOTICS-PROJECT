  • Robotics wheel controlled via bluetooth

    how it works

    The robotics wheelchair's control
    system makes use of the Arduino
    microcontroller and HC-05
    Bluetooth module,
    which are connected via bluetooth to smartphone app called BT car.
    The Arduino interprets the control signals
    sent by the user through
    the app,which include stopping,going left, right and forward.
    The fundamental algorithm interprets
    the instruction from bluetooth module
    and sets the proper motors to run
    at predetermined rates
    The wheel chairs motors change
    direction and speed in response to input, which include
    forward, backward,left and right.

  • Project output

    The outcome of the project ia a
    functioning robotic wheelchair that
    can be controlled wirelessly via
    a smartphone.
    The wheel chair responds to user input
    sent through the BT Car app,
    allowing the user to navigate the
    wheel chair in all directions
    It offers precise control,allowing
    users to move easily in tight spaces
    and achieve smoother, more
    flexibility.
    The system is also cost effective
    , using readily available components.

    Applications

    Home
    Hospitals

    RAPID RESCUE ROBOT USING RASPBERRY PI

    how it works

    The rapid rescue robot enables
    rescue crews to remotely control
    and survey the sorroundings. It is made for speedy
    deployments in
    disaster regions.The operator
    connects to a hotspot, uses puTTY to contact
    the robot, and then copies the IP adress of
    the robot into
    the browser
    to initiate a control interfaces.
    The operator can see a live video feed back from robots
    camera and control its movements
    with this interface.
    Like a radar, the robot scans its environment
    with its ultrasonic sensors to
    find gaps and obstructions among the
    debris. Real time
    data visualisation aids
    rescue crew in pinpointing
    vital locations where the
    possible survivors may be found
    After that, the robots operator can direct
    it to specific spots, greatly reducing the
    search area and enabling rescuers to concetrate
    their efforts.
